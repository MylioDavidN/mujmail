Provides (user) folders functionality. Currently takes care about user folders.

Holds list of user folders, creates and loads theirs databases. User folders are implemented as InBox instances. Theirs names are stored in special database. Each user folder has (and also loads and saves) list of account's to retrieve. See {@link mujmail.InBox#saveBoxRetrieveAccounts}

Note: Classes {@link mujmail.TheBox}, {@link mujmail.PersistentBox}, {@link mujmail NonPersistentBox}, {@link mujmail.InBox}, {@link mujmail.Trash}, {@link mujmail OutBox} should be refactored into this package.
