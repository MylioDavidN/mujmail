Provides capability for storing and viewing debug information on mobile device. By default information are copied to standard output and stored in vector to be able display them to user.

Can store information persistently in separate database.

  • DebugConcole class is used for writing debug information. Provides line string oriented interface for programmers to print debug in information.
  • DebugConsoleUI shows debug information in special form.

To print debug log call {@link mujmail.debug.DebugConsole#println(String)}

Note: Persistent storing of debug information is usefull for on device debugging if application falls.
