Contains unit tests.

Note, that it is not possible to run unit tests when obfuscation level is setted to high (tested on JDK6 and WTK 2.5.2).

To compile unit tests, add library lib/j2meunit.jar to the classpath.  To run tests, run midlet unit_tests/TestingMidlet (in netbeans add the midlet to the configuration of your project:  Application  descriptor -> MIDlets -> Add), the project and choose testing middlet.

For more information about this library look there: j2meutni Nice tutorial how to write unit tests is there.
